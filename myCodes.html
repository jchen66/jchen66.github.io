Interactive Game

  The Interactive Board Game uses Processing libraries to build a virtual board on screen that the user can tilt by dragging the mouse. It can also be controlled by rotating a lego board in front of the camera.
The goal of the game is to not let the ball drop out of the board, nor let it bump into the cylinder obstacles for as long a possible! The challenging part of this project is that the virtual board can also be controlled by placing a board with vibrant colors in front of the camera and tilting it. Our code will use blob detection and Hough line detection to apply to movement to the virtual board.

Built with: processing, Blender.

Parser Evaluator

 Using regular expressions, a parser takes in a string(WML inspired language) and then returns the root of a tree object(AST node). It can then evaluate the string by taking in the root AST node as a parameter.
The function can then evaluate simple "#if", "#ifeq" and "#expr" invocations and perform the appropriate behaviour.

Built with: javascript.
